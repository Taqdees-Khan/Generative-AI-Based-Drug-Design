# Supporting Information

This directory contains supporting information files for the JCIM publication.

## Files:
- `Table_S1_binding_site_residues.csv` - EGFR binding site residue analysis
- `Table_S3_plip_interactions.csv` - PLIP interaction analysis results
- `Figure_S1_rmsd_analysis.png` - Erlotinib re-docking RMSD analysis
- `Figure_S2_binding_affinity_correlation.png` - Binding affinity predictions

## Description:
These files provide detailed supplementary data referenced in the main manuscript.
All analyses were performed using the BindingForge pipeline with validated parameters.
